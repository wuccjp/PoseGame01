# Web 原型：手掌偵測 + 實心圓覆蓋 (Vanilla JS MVP)

本專案提供一個 Web 原型，用於在攝影機影像中偵測手掌，並在手掌上方顯示實心圓。左手顏色為黃色、右手顏色為紅色；畫面以 Canvas 覆蓋在影片之上，同時實現左右影像鏡像呈現的效果。

核心特性
- 實時手掌偵測（最多兩隻手）
- 在手掌中心點上方顯示實心圓，圓形大小固定為畫面高度的 1/10，半徑為高度的 1/20
- 左手與右手顏色映射：黃色與紅色
- 影像鏡像顯示（左右顛倒），圓形位置以鏡像後的座標對應
- 介面提供顯示/隱藏影像的切換控制
- 本地處理、不上傳影像（隱私友好設計）

架構概覽
- index.html: 應用入口，包含 video、canvas、控制面板
- styles.css: 版面與樣式（含鏡像設定與控制面板樣式）
- main.js: 核心執行邏輯，負責相機、MediaPipe Hands 整合、圓形繪製與渲染迴圈
- config.js: 設定參數（未在本版本嚴格使用，方便日後調整）
- libs/mediapipe/hands/: MediaPipe Hands JS 套件
- modules/: 模組化的前端模組（camera.js、handDetector.js、overlay.js 等，於目前 MVP 專案中以單檔主控為主）

快速開始
1) 取得專案檔案
- 將 index.html、styles.css、main.js、config.js、README.md、以及 test/ 資料夾下的檔案放在同一工作目錄。
2) 啟動本地伺服器
- 使用 Node.js 的 http-server（或 Python 皆可）在專案根目錄啟動本地伺服器。
  - Node.js (需先安裝 Node.js):
    - npx http-server -p 8080
  - Python:
    - python -m http.server 8000
3) 開啟瀏覽器
- 進入 http://localhost:8080（或 8000，視你啟動的埠而定）
4) 相機授權
- 首次載入時，瀏覽器會請求允許使用相機，允許後即可在畫面看到實時影像，並於覆蓋層看到手掌上的實心圓。

如何修改與進階
- 圓形大小：目前固定為高度的 1/10，半徑為高度的 1/20。可在 main.js 調整或在 config.js 設定後導入。 
- 顏色與左右映射：目前 Left 對應黃色、Right 對應紅色，鏡像顯示後仍以 centerX = width - center 為對應，以確保圓形落在鏡像後的手掌位置。若需更改映射邏輯，可在 renderOverlay 函式中修改顏色對應規則。
- 影像顯示切換：UI 右上方新增的按鈕可切換顯示/隱藏攝影機影像，遮罩層仍可見偵測結果。
- 後續擴充：可加入平滑濾波、信心閾值、多圈圓形、分數機制等，並將核心模組抽成可重用的組件。

注意事項與隱私
- 本專案在本地端運行，影像不會上傳至伺服器，僅在使用者裝置上完成偵測與繪製。
- 如需公開部署，請留意相機授權與使用者知情同意。
